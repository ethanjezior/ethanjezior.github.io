Hexadecimal, RGB, and HSL are all similar yet uniquely different. What I mean by this is that while the values for the three are each different the same color values are available across all three, functionally making them three paths that lead to the same end location.

Preferred color code format: Hexadecimal is my preferred color code format due to it being the simplest for me to type rapidly.

HWB summary: HWD color uses the three values (hue, whiteness, blackness) to asign colors based on a three input code similar to that of rgb or hexadecimal.

CMYK summary: CMYK uses four values(as opposed to three) to make up its color pallet.
