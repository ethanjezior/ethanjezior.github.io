Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

The sticky position is useful for making responsive web pages that adjust to be as pleasant as possible regardless of the platform viewed on.

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

The relative position is useful for making an elemnts position and/or size change based on the elemnts position relative to something else, such as the top or left sides of the screen(viewport).

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

The fixed position is used to make an element stay glued to a spot regardless of whether or not the user scrolls.

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

An element using the absolute position is positioned relative to the nearest positioned ancestor, as appossed to positioned relative to the viewport.
